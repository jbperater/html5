Header

Defines a header for the document or a section

The < header> element represents a container for introductory content or a set of navigational links.

one or more heading elements (<h1> - &lh6>)

Nav

Defines navigation links in the document

Notice that NOT all links of a document should be inside a <nav> element. The <nav> element is intended only for major block of navigation links.

Browsers, such as screen readers for disabled users, can use this element to determine whether to omit the initial rendering of this content.

Aside

Defines content aside from the page content

The aside content should be related to the surrounding content.

Article

Defines an article in the document

An article should make sense on its own and it should be possible to distribute it independently from the rest of the site.

Potential sources for the <article> element:

Forum post

Blog post

News story

Comment

Section

Defines a section in the document

The <section> tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.

Time

Defines a date/time

The <time> tag defines a human-readable date/time.

This element can also be used to encode dates and times in a machine-readable way so that user agents can offer to add birthday reminders or scheduled events to the user's calendar, and search engines can produce smarter search results.

Footer

Defines a footer for the document or a section

A <footer> element should contain information about its containing element.

A <footer> element typically contains:

authorship information

copyright information

contact information

sitemap

back to top links

related documents

You can have several

elements in one document.
